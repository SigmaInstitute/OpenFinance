OpenFinance

An open, community-built benchmarking platform for Implementable Asset Pricing and Covariance Forecasting.

This repository is an MVP website that can be hosted on GitHub Pages. The long-term goal is a fully automated model-to-data evaluation system with a private holdout test set.

Vision

Bridge the gap between academia and practice by enabling apples-to-apples comparisons of models under the same data, same preprocessing, and a more realistic evaluation protocol.

OpenFinance aims to become shared research infrastructure: a stable protocol, transparent metrics, reproducible submissions, and a public leaderboard—so results are comparable and cumulative.

Why OpenFinance

• Fair comparison: standardized dataset schema, splits, and scoring.

• Realism: implementability-first evaluation (liquidity constraints & transaction costs).

• Reproducibility: containerized, sandboxed runs with pinned dependencies and run logs.

• Open collaboration: non-exclusive, multi-institution governance and contribution-based credit.

See Governance for roles, RFC changes, and credit.

Tracks

Implementable Asset Pricing Participants submit periodic portfolio weights. The platform evaluates net-of-cost performance under a standardized liquidity and transaction cost model.

Covariance Forecast Participants submit covariance forecasts. The platform evaluates statistical losses and decision-aware risk outcomes.
