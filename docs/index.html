Online Benchmark Demo

A fastest-possible prototype that mimics a “Common Task Framework” leaderboard site: rules → dataset access → submissions → leaderboard. Designed to be hosted on GitHub Pages.

Tracks (Demo)

Implementable Asset Pricing Submit monthly portfolio weights. The platform scores net-of-cost performance under a standardized liquidity constraint and transaction cost model (in the full system).

Covariance Forecast Submit next-period covariance forecasts. The platform evaluates statistical loss and decision-aware utility (in the full system).

Hosting: GitHub Pages Data feed: data/leaderboard.json Stack: HTML + CSS + JS

What this demo includes

• A clean leaderboard page reading a JSON file.

• Rule pages & dataset access page (static text).

• A submission workflow suggestion (GitHub PR/Issue, no backend).

In the real platform, submissions are executed in a network-disabled sandbox and scored on a private holdout test set.
